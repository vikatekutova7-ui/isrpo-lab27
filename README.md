## 🛣️ Список реализованных маршрутов

| Метод | Маршрут | Описание | Параметры | Возвращаемое значение |
|-------|---------|----------|-----------|----------------------|
| `GET` | `/` | Приветственное сообщение | — | `string`: "Hello World!" |
| `GET` | `/users` | Получение списка пользователей | — | `JSON[]`: массив пользователей |
| `GET` | `/users/{id}` | Получение пользователя по ID | `id` (int) | `JSON`: объект пользователя или `404 Not Found` |
| `POST` | `/users` | Создание нового пользователя | `User` (body) | `JSON`: созданный пользователь + статус `201 Created` |
| `PUT` | `/users/{id}` | Обновление пользователя | `id` (int), `User` (body) | `JSON`: обновлённый пользователь или `404` |
| `DELETE` | `/users/{id}` | Удаление пользователя | `id` (int) | `204 No Content` или `404` |
| `GET` | `/health` | Проверка работоспособности | — | `JSON`: `{"status": "ok"}` |

### 💻 Пример реализации маршрута:
```csharp
app.MapGet("/users/{id}", (int id, IUserService service) =>
{
    var user = service.GetById(id);
    return user is null ? Results.NotFound() : Results.Ok(user);
});
